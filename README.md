# Doodlepad
Final project for Technopreneurship course.

## App Overview
Doodlepad is a mobile drawing application for kids.

```Features``` :wrench:
* custom color pallet
* custom brush size
* gallery
* Undo
* Redo
* erase
* save button
* color picker
* share feature to social media site
* add custome background
